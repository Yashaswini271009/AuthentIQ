Privacy & Detection Ethics

  • We use **official APIs** where possible (e.g., Email OAuth, Twitter Bearer Token).
  • All scraping (if used) respects **robots.txt** and user consent.
  • No login credentials or private messages are ever stored.
  • Detection is done via **public metadata, follower ratios, and posting behavior**.
  • Data sent to backend is **encrypted**, and processed temporarily.
  • We follow academic and ethical standards for machine learning classification.
